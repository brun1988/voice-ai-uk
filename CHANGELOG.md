# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-17

### Added
- Full-stack Next.js 14 application with App Router
- NextAuth.js authentication (credentials + GitHub)
- Prisma ORM with Supabase PostgreSQL
- Agent builder with flow designer UI
- Twilio phone integration (+44 UK numbers)
- Vapi voice AI integration
- Phone number purchase and management API
- Call logging and history
- Analytics dashboard with charts
- User and business settings
- Complete landing page
- Production-ready Vercel configuration
- Comprehensive documentation (README, DEPLOYMENT.md, DOMAIN_SSL.md, QUICKSTART.md)

### Features
- Multi-tenant architecture (teams/workspaces)
- Real-time voice webhooks
- Agent routing configuration
- Knowledge base upload support

### Tech Stack
- **Frontend:** Next.js 14, React, Tailwind CSS
- **Backend:** Next.js API Routes, Prisma
- **Database:** PostgreSQL (Supabase)
- **Auth:** NextAuth.js
- **Voice:** Vapi, Twilio
- **Deployment:** Vercel
